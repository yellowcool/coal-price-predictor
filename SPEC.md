# 煤价预测系统规格说明

## 1. 项目概述

- **项目名称**: 煤价预测系统
- **类型**: Web 全栈应用（前后端分离）
- **核心功能**: 单页面展示所有煤种实时价格、历史趋势及预测数据
- **目标用户**: 煤炭行业从业者、分析师

## 2. 技术栈

- **前端**: Vue.js 3 + Vite + ECharts + TailwindCSS
- **后端**: Flask (Python)
- **API 通信**: RESTful JSON

## 3. UI/UX 规格

### 3.1 布局结构

- **顶部**: 标题栏（固定高度 64px）+ 数据更新时间
- **主体**: 4 列煤种卡片网格（桌面），移动端 1 列
- **卡片间距**: 24px
- **响应式断点**:
  - Mobile: < 768px (1 列)
  - Tablet: 768px-1024px (2 列)
  - Desktop: > 1024px (4 列)

### 3.2 视觉设计

- **主色调**: 
  - 主色: #1E3A5F (深蓝)
  - 强调色: #F59E0B (琥珀)
  - 背景: #F8FAFC (浅灰白)
  - 卡片背景: #FFFFFF
- **字体**: 
  - 标题: 24px, font-weight: 700
  - 卡片标题: 18px, font-weight: 600
  - 数据: 16px, font-weight: 500
- **阴影**: 0 4px 6px -1px rgba(0,0,0,0.1)
- **圆角**: 12px

### 3.3 煤种卡片组件

- **当前旬均价**: 大字体突出显示 (28px, 粗体)
- **上/中/下旬价格**: 三列布局
- **最高/最低价**: 标签形式展示
- **环比变化**: 涨（红色↑）跌（绿色↓）百分比
- **趋势图表**: ECharts 折线图，显示 12 个月历史 + 3 旬预测

## 4. 数据规格

### 4.1 煤种及价格范围

| 煤种 | 价格范围 (元/吨) | 波动范围 |
|------|------------------|----------|
| 焦煤 | 850-950 | ±5% |
| 肥煤 | 900-1000 | ±5% |
| 1/3 焦煤 | 800-900 | ±5% |
| 瘦煤 | 750-850 | ±5% |

### 4.2 数据结构

```json
{
  "coalTypes": [
```

## 5. API 接口

- `GET /api/all-data` - 返回所有煤种完整数据
- `GET /api/coal-types` - 返回煤种列表
